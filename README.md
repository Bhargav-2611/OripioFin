# 💰 OripioFin Dashboard UI

## 📌 Project Overview
The **Finance Dashboard UI** is a responsive frontend application built using **HTML, CSS, and JavaScript** to help users track and understand their financial activities.

This project focuses on:
- Clean UI design  
- Interactive components  
- Data visualization  
- Efficient state handling using JavaScript  

---

## 🎯 Objectives
- Provide a clear financial summary  
- Enable easy transaction management  
- Visualize spending patterns  
- Demonstrate frontend development skills without backend dependency  

---

## 🚀 Features

### 🧾 Dashboard Overview
- Summary cards:
  - Total Balance  
  - Total Income  
  - Total Expenses  
- 📈 Time-based chart (balance trend)  
- 📊 Category-based chart (expense distribution)  

---

### 💳 Transactions Section
Displays transaction details:
- Date  
- Amount  
- Category  
- Type (Income / Expense)  

**Functionalities:**
- 🔍 Search transactions  
- 🎯 Filter by category/type  
- 🔃 Sort by date or amount  

---

### 👥 Role-Based UI (Simulated)
- **Viewer**
  - View-only access  
- **Admin**
  - Can add/edit transactions  

➡️ Role switching is implemented using a UI toggle/dropdown.

---

### 📊 Insights Section
Provides useful financial insights:
- Highest spending category  
- Monthly comparison  
- Basic analytics from transaction data  

---

### 🧠 State Management
- Managed using **Vanilla JavaScript**  
- Handles:
  - Transactions data  
  - Filters and search  
  - Role switching  

---

## 🎨 UI & UX Highlights
- Responsive design (mobile + desktop)  
- Clean and minimal layout  
- Dark mode support (if implemented)  
- Smooth interactions and modals  
- Handles empty/no data states
